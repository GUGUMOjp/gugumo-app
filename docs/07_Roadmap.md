# GUGUMO Roadmap

Version: 1.0

Status: Active

---

# RM-001 目的

本ロードマップは、GUGUMOの開発・事業・技術の中長期計画を定義する。

目的は以下の通り。

- 開発の優先順位を明確にする
- 機能追加の順序を管理する
- 将来の事業拡大を見据えて設計する
- 開発判断の基準とする

---

# RM-002 開発方針

GUGUMOは

**小さく作り、大きく育てる**

を基本方針とする。

優先順位は

運用改善

↓

保守性

↓

利益率

↓

機能追加

とする。
